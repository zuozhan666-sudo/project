# 旅·分享（纯静态示例）

这是一个纯静态的旅游分享作业模板（HTML/CSS/JS），包含以下页面：
- index.html（首页）
- track.html（城市轨迹，使用 Leaflet + OpenStreetMap）
- photos.html（风景照片，相册支持本地上传并保存到 localStorage）
- videos.html（视频分享，支持外链）
- food.html（美食分享，发布到 localStorage）
- guides.html（游玩攻略，发布到 localStorage）
- assets/ 包含示例图片（SVG）

运行方式：
1. 将文件放到同一目录（或放到 GitHub 仓库的 docs/），
2. 直接用浏览器打开 index.html 即可（无需安装）。
3. 或者把仓库启用 GitHub Pages（选择 GitHub Pages 来源为 main branch /docs folder，若你放在 docs/ 下）。

数据说明：
- 本演示为纯前端 demo，所有用户上传的数据都保存在浏览器 localStorage（用于作业演示）。
- 若要真实部署并保存文件，请后续接入后端或第三方存储（超出当前作业范围）。
